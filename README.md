# Password Strength Checker - Task 03

This project is a Python-based tool that evaluates the strength of a password based on multiple security criteria.

## Features
- Checks password length
- Checks for most common passwords
- Validates uppercase and lowercase characters
- Ensures presence of numbers and special characters
- Detects repeated characters
- Warns against using spaces
- Provides feedback for improvement

## How It Works
The program analyzes the password and assigns a strength level:
- Weak
- Moderate
- Strong

It also provides suggestions to improve weak passwords.
